# Gestor Financiero Personal

Aplicación de gestión financiera desarrollada con **Flet (Python)** para el curso de Fundamentos de Aplicaciones Móviles.

## Funcionalidades
- Dashboard con resumen financiero
- Gestión de ingresos y gastos
- Categorización de transacciones
- Gráficos interactivos (barras y torta)
- Sistema de alertas por presupuesto
- Exportación a Excel y PDF
- Temas claro/oscuro

## Instalación y Ejecución

1. **Clonar el repositorio:**
```bash
  https://github.com/YX-RY/App_Bancaria.git
```
2. **Instalar dependencias:**
```bash
  pip install -r requirements.txt
```
3. **Ejecutar la aplicación:**
```bash
  python main.py
```
